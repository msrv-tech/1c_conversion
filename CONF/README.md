# CONF

Эта папка содержит конфигурационные файлы и маппинги.

В текущем публичном репозитории уже включены данные для основного сценария миграции:

- источник: `УПП`
- приемник: `УХ`

Здесь хранятся:

- метаданные конфигураций
- JSON-маппинги
- база `type_mapping.db`

Если репозиторий будет использоваться для другой пары конфигураций 1С, содержимое этой папки нужно заменить или дополнить собственными файлами под новый проект.
